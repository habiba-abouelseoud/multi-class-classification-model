# Coin Classification & Segmentation Project

This project implements a complete computer vision and deep learning pipeline to **detect, segment, and classify coins from images**. The system performs end-to-end analysis, from raw images to structured insights and dataset-level statistics.

## ğŸš€ Project Capabilities

The pipeline is capable of:

- Segmenting coins from images using a trained **instance segmentation model**
- Classifying coins as **UK or non-UK currency**
- Identifying **Heads vs. Tails** orientation
- Determining the **denomination of UK coins**
- Performing **full dataset analysis** to answer predefined analytical questions

### ğŸ“ Notebooks/
Contains all development and experimentation notebooks used throughout the project.

- **Multi-output_Coin_Classifier.ipynb**  
  Trains a **multi-output classification model** that predicts:
  - UK vs. Non-UK coin  
  - Coin side (Heads or Tails)  
  - Denomination of UK coins  

- **GUI_For_DataLabeling.ipynb**  
  A helper notebook providing a simple interface to assist with **manual data annotation** for classifier training.

- **Coin_Segmentation_Model.ipynb**  
  Trains an **instance segmentation model using Detectron2** to detect and segment coins from raw images.

- **Full_Pipeline_Dataset_Questions.ipynb**  
  Executes the **entire pipeline end-to-end**, including:
  - Coin detection and segmentation  
  - Coin classification  
  - Visualization and statistical analysis to answer dataset questions such as:
    - Distribution of total coin values per image  
    - Percentage of Heads vs. Tails  
    - Coin counts and denomination frequencies
